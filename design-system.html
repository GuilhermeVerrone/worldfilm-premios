Breakpoints

Pontos de quebra do layout responsivo
Breakpoint Mudanças principais
≤ 1600px .topoLinks fica absolute (position: absolute; top: 0; right: 0). .menuFundo margin: 0.
≤ 1200px .conteudo padding 0 20px. Imagem do banner começa a ampliar (width: 140%, margin: 0 -20%).
≤ 1024px .conteudo vira 100%. .bloco ganha padding lateral 15px. .atalho vira 100%. .produtoItem vira 100%. Colunas .formCol3 viram 33%.
≤ 900px Banner 160% de largura. Links 2 e 3 do topo ocultam texto (só ícone). Ajustes no simulador.
≤ 700px Menu hamburger. .topoLinks oculto. .menuFundo vira drawer lateral (250px, slide da direita). .subtitulo vira 6vw. Colunas .formCol3–10 viram 100%.
≤ 600px Banner 200% de largura.
≤ 480px Ajustes do popup: padding reduzido, botões em coluna.
≤ 400px .formCol3 (grid de produtos) vira 100%.
